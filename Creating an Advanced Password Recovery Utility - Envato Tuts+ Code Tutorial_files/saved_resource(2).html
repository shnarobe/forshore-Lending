                                Can we all please agree sha256 > md5?

                                While I love the idea of this tutorial you way over complicated the problem. I recently added a Forgot Password option to my user class and it is under 20 lines of code.

                                what is entered on http://yourdomain.tld/password/reset to $user->getAuthToken().
